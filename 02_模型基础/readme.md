## 梯度消失

链式求导法则中，连乘时都是小于1的数据，结果将越来越小。梯度消失只能尽量减轻，无法杜绝。

手段：
1、Relu，激活函数。将上一层神经元相乘相加的结果拿到汇总，经过变换后，
拿到的结果一定是大于零的。

收敛：连续十次~二十次 变化幅度不大
