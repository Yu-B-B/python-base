# 进阶

## 闭包

闭包是指在一个外部函数中定义的内部函数，该内部函数引用了外部函数的变量，
并且外部函数返回这个内部函数。这样，即使外部函数执行完毕，内部函数仍然可以访问和记住外部函数的变量。
